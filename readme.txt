== Description ==

Auto Meta Header plugin automatically add meta keywords and description on your WordPress blog, Also Friendly Search Engine Robot Meta Tag. The title of Frontpage/single post/tag/category/else page set as the main keyword for each of it. Recommended to adding some target keyword and description for Frontpage/single post/tag/category/else page by Customize Auto Meta Header Option at wp-admin dashboard setting menu.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `auto-meta-header.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Setting -> Auto Meta Header and customize by inserting your target keyword and description, or leave it blank for default mode.

= How do I add description manually? =
Type description on the textbox and recommended is more than 100 chars and maxlength is 300 Chars
